## Instalacja

1. Trzeba zaintalować fonty z folderu `pismo/.fonts`.
1. Trzeba zainstalować system XeLaTeX.

## Użycie
1. Wystarczy skompilować plik `pismo.tex` poleceniem `xelatex pismo.tex`.
